# Bear-Service-Site
Site estático personalizado para uma barbearia

Projeto demo: Site estático para uma barbearia

Arquivos e instruções rápidas

- Páginas: `index.html`, `services.html`, `gallery.html`, `about.html`, `contact.html`, `booking.html`
- Estilos: `assets/css/styles.css` — altere as variáveis CSS em `:root` para ajustar as cores
- Imagens: coloque arquivos em `assets/images/` e atualize os elementos nas páginas

Agendamento (Calendly)

- Em `booking.html` substitua `https://calendly.com/SEU_USUARIO` pelo link da sua conta Calendly.
- Para sincronizar com Google Calendar, conecte sua conta nas configurações do Calendly (o Calendly cuidará das confirmações e sincronização).

Formulário de contato

- O formulário em `contact.html` aponta para Formspree (`action="https://formspree.io/f/YOUR_FORM_ID"`). Substitua `YOUR_FORM_ID` pelo ID do formulário do seu serviço preferido.

Deploy no Vercel (rápido)

1. Faça login no Vercel (https://vercel.com) e crie um novo projeto.
2. Aponte para este repositório e aguarde o deploy automático.
3. O site estará disponível em `*.vercel.app` — conecte seu domínio nas configurações do projeto.

Personalização e próximos passos

- Substitua textos e imagens pelas suas informações reais
- Se quiser migrar do widget para um agendamento custom, podemos planejar uma API serverless posteriormente
